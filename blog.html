Hey there, tech enthusiasts! It's day 3 of my journey into the world of web development, and I'm thrilled to share my progress with you. As a 40-year-old manager leading a team in a tech support call center, diving back into HTML, CSS, and Javascript feels like a refreshing refresher after two decades. Getting My Hands Dirty with HTML:
I've just wrapped up the HTML section, and it's incredible how the basics remain the foundation of web development. Creating my first website in class was a nostalgic trip down memory lane. I'm planning to evolve it as I progress through the lessons, adding layers of complexity and functionality.

The Tech Stack I'm Building:

As I embark on this journey, I'm delving into both front-end and back-end technologies. Here's a snapshot of what I willing be learning in this course:

Front-End:

  • HTML5: The backbone of web content.
  • CSS3: Adding style and aesthetics.
  • Javascript: Making things interactive.
  • React: Building dynamic user interfaces.

Back-End:

  • Node Server: Enabling server-side scripting.
  • Node.js: The runtime for executing Javascript on the server.
  • Express.js: Simplifying the process of building web applications.
  • PostgreSQL: My relational database of choice.
  • MongoDB: Exploring the world of NoSQL databases.

Reflecting on Past Technologies:

In my previous tech stack, HTML, CSS, Javascript, and JQuery were the building blocks for the front-end. On the back-end, the LAMP stack (Linux, Apache, MySQL, PHP) was the go-to setup. It's fascinating to see how the landscape has evolved over the years, with new technologies offering enhanced capabilities and efficiency.

Navigating the World of Forms:

Understanding forms and their intricacies is crucial. I've dived into the query string, different methods (GET, POST), and actions on the backend (like action.php).

Looking Ahead:

With a renewed passion for coding, I'm excited about the possibilities that lie ahead. I aim to become a full-stack developer by the end of the year, armed with a comprehensive skill set that spans both front-end and back-end technologies.
